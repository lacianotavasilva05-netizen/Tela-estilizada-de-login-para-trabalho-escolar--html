Este projeto consiste no desenvolvimento de um site com foco em simular um sistema de transporte escolar, apresentando uma interface moderna e organizada. A aplicação conta com tela de login estilizada, identidade visual personalizada e estrutura bem definida em pastas, seguindo boas práticas de desenvolvimento web.

O objetivo principal foi aplicar conceitos de HTML e CSS para criar um layout funcional, visualmente agradável e adaptável a diferentes dispositivos, demonstrando organização, planejamento e atenção ao design.
